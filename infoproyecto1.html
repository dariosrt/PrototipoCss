INFORMACIÓN 1º PROYECTO

HTML no es un lenguaje de programación, sino un lenguaje de marcado, ya que sirve para definir, organizar y presentar el contenido de una página web usando etiquetas.

No tiene la capacidad de ejecutar lógica o crear funcionalidades dinámicas, sino que se enfoca en estructurar la información.
HTML fue creado por Tim Berners-Lee en 1989 en el CERN para compartir información entre científicos a través de la web. La primera versión oficial apareció en 1993, y desde entonces ha evolucionado para mejorar la estructura y funcionalidad de las páginas. Algunas actualizaciones como HTML 2.0 (1995) y 3.2 (1997) añadieron nuevas etiquetas y soporte para tablas, mientras que HTML 4.01 (1999) organizó mejor el contenido. En 2014, llegó HTML5, la versión moderna que permite integrar video, audio y aplicaciones interactivas, consolidando a HTML como la base de toda la web.
