CSS significa Cascading Style Sheets o Hojas de Estilo en Cascada. Es el lenguaje que se usa para dar estilo y presentación a las páginas web creadas con HTML. CSS es un lenguaje de hojas de estilo. Esto significa que no programa acciones ni lógica, sino que describe cómo se debe presentar el contenido de una página web.
CSS fue propuesto por Håkon Wium Lie en 1994 para separar el contenido de la presentación en la web. En 1996, el W3C publicó CSS1, con propiedades básicas de fuentes, colores y márgenes. En las siguientes versiones se añadirían propiedades como posicionamiento avanzado, bordes redondeados, sombras...
